Treinamento em Arquitetura e
Desenvolvimento de Software

Objetivo:

O objetivo deste treinamento é preparar uma Equipe de Desenvolvimento de software profissional com as mais modernas técnicas de desenvolvimento e arquitetura de software, visando em especial a qualidade no processo de desenvolvimento e alinhamento com novas tecnologias, frameworks e melhores práticas.

Metodologia:
Aulas remotas, incluindo listas de exercícios com correções das mesmas e publicação da solução em repositorio público. Desenvolvimento de projeto prático, normalmente um projeto real em que a equipe de desenvolvimento trabalhara em conjunto com o instrutor vivenciando o dia-a-dia do desenvolvimento e onde aparecem necessidades de melhoria da arquitetura e novas problemas a serem resolvidos.

Ferramentas

* Java 8 Update 261
* Git for Windows
* IntelliJ Community
* Player WRF
Apache Maven (opcional)
- Descompacte o Zip em uma pasta de sua preferência
- Adicione a pasta <Maven Home>\bin no PATH do Sistema Operacional
- Configure o IntelliJ para utilizar esta instalação de Maven
nssm (install windows service)
